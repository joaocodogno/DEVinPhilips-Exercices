      MAIS INFORMAÇÕES SOBRE DEV IN KNOWLEDGE

      Esta aplicação permite o cadastro de dicas relacionadas a área de tecnologia.

      Para cadastrar uma dica, utilize o formulário da direita e preencha os campos título, linguagem, categoria e descrição (o campo vídeo é opcional, no entanto aceita apenas URL).

      Ao cadastrar uma dica, ela aparecerá na área de visualização a esquerda.

      Para filtrar uma dica, utilize os cards para filtrar por categoria ou a barra de busca para filtrar pelo nome.

      Para limpar os filtros, clique no botão 'refresh' ao lado do botão pesquisar

      Para editar uma dica já cadastrada, clica no ícone editar no canto inferior direito da dica a ser editada.

      Para deletar uma dica cadastrada, clique no 'X' localizado no canto superior direito de cada dica.
